#### How to run vroom
Code editor and node required
1. Clone down this repo using your terminal
2. `npm i`
3. `npm run knex migrate:latest`
4. `npm run knex seed:run`
5. `npm run dev`
6. In your browser, run 'http://localhost:3000/'
